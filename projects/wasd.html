Wasd

Wasd is a program I wrote for my Programming Languages course. The program uses a digital differential analyzer algorithm to help draw pixelated lines on an image to effectively walk a gridded world looking for collusion with rays. I wrote this program in Java using threading and Lambdas. The program generates a navigable 3D map similar to games such as Doom or Wolfenstein.
