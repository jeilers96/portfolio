coOperation: Breakout

Tools Used:

  • C#
  • Unity Game Engine
  • GIMP
  • Aurdino Uno

coOperation: Breakout is a cooperative two-player puzzle platformer. Play as a group of science experiments working together to break out of a lab that has been preforming inhumane tests on them. Navigate through puzzles using each science experiments' unique abilities and work together as a team to achieve the taste of freedom.

Braden Lahr, Nick Pientka, and I created coOperation: Breakout using the Unity game engine for our Game Design course. The goal of the course was to work as a team to create a two-dimensional cooperative arcade game. We were required to create our own hardware to control the game as well. I worked on all aspects of this game from creating the art to programming many of the game mechanics. I also programmed the Arduinos to take in user input from arcade buttons and send it as input into the game. A playthrough of the game can be seen here, and the code for the game is presented in this GitHub repository.
